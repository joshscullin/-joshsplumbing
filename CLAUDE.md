## About this project
I am a beginner learning to code by building real projects.

This project started as a landing page for Josh's Plumbing and is being expanded into a fuller business system.

## How to work with me
- Explain every important decision in simple terms.
- Plan before building.
- Break work into clear steps.
- Teach as you go, not just output code.
- Assume I may not know the terminology yet.
- When making changes, explain what changed and why.

## Technical preferences
- Do not use Computer Use.
- Use plain HTML, CSS, and JavaScript unless I explicitly agree to a framework.
- Ask me before choosing or changing the tech stack.
- Prefer simple solutions over advanced abstractions.
- Keep code beginner-friendly and readable.

## Current setup
- I am currently using sql.js.
- The site stores contact information submitted through the website.
- There are 2 main pages:
  - `localhost:3000` = landing page
  - `localhost:3000/admin` = admin page showing lead information

## Coding rules
- Do not introduce frameworks unless I approve them first.
- Do not make major architecture changes without explaining the tradeoffs first.
- Before writing larger features, provide a short plan.
- Keep files and structure simple unless complexity is clearly needed.
- Comment code when it helps learning.

## When helping
- Tell me what file you are changing.
- Tell me why the change is needed.
- Tell me how I can test it.
- Flag anything that is temporary, fragile, or not production-ready.
